# YeMax - Неофициальная модификация мессенджера MAX

YeMax — мощная неофициальная модификация популярного мессенджера MAX с расширенными возможностями для продвинутых пользователей.

[![Telegram](https://img.shields.io/badge/Telegram-YeMaxMod-blue?logo=telegram)](https://t.me/YeMaxMod)

## Возможности

- **Две версии**:
  - Чистая версия (минималистичная)
  - Версия с фишками (расширенный функционал)

- **Анти-удаленки** — восстановление удаленных сообщений
- **Меню разработчика** — доступ к скрытым настройкам
- **Удаленная слежка и аналитика** — мониторинг, трекеры, статистика

## Установка

1. Скачайте APK из [Telegram-канала](https://t.me/YeMaxMod) или GitHub Releases
2. Разрешите установку из неизвестных источников
3. Установите и запустите

**⚠️ Внимание**: Неофициальная модификация. Используйте на свой страх и риск.
